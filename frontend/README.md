

# Xeplin Challenge Frontend


Features:
`Next`, `Redux Toolkit`, `Shadcn`, `Tailwind`, `Eslint` and `Jest`

## Start APP locally

- _Access the folder_

```
cd xepelin-challenge/frontend

```

- _Install the dependencies_

```
npm install
```

- _Run the project_

```
npm run dev
```

- _Run Test_

```
npm run test
```


- _Run Test coverage_

```
npm run test:coverage
```
