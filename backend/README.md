# Xeplin Challenge Backend

Features:
`Nest`, `CQRS`, `Postgres`, `Mongo`, `Eslint` and `Jest`


## Start APP locally

- _Access the folder_

```
cd xepelin-challenge/backend

```

- _Install the dependencies_

```
npm install
```

- _Run the project_

```
npm run start:dev
```

- _Run Test_

```
npm run test
```

- _Run Test coverage_

```
npm run test:cov
```
