# xepelin-challenge

## Requirements
- git
- Node
- npm
- Docker

- Features:
`Nest`, `CQRS`, `Postgres`, `Mongo`, `Next`,
`Redux Toolkit`, `Shadcn`, `Tailwind`, `Eslint` and `Jest`


## Start APP locally

- _Access the folder_

```
cd xepelin-challenge/

```

- _Run the project_

```
npm run services:init
```

- Stop the project_

```
npm run services:stop
```
To run the app:
```bash
backend http://localhost:4000
frontent http://localhost:3000
```
